# fzf-cli
> https://github.com/fuzhonfeng/fzf-cli


## Install

```
npm install fzf-cli -g --registry=https://registry.npm.taobao.org
```

## Command
### init project 创建模板项目

```
fzf init
```

### run project 启动项目(***需要在项目根目录执行***)

```
fzf run
```

### publish project 打包项目(***需要在项目根目录执行***)

```
fzf pub
```

### lint project 代码风格校验(***需要在项目根目录执行***)

```
fzf lint
```
