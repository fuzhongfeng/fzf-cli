# axx-cli
> https://github.com/gaosife/axx-cli


## Install

```
npm install axx-cli -g --registry=https://registry.npm.taobao.org
```

## Command
### init project 创建模板项目

```
axx init
```

### run project 启动项目(***需要在项目根目录执行***)

```
axx run
```

### publish project 打包项目(***需要在项目根目录执行***)

```
axx pub
```

### lint project 代码风格校验(***需要在项目根目录执行***)

```
axx lint
```
